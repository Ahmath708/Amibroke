05

Motion

↺Press feedback on every tappable element — a quick scale / spring on tap.
◷Continuous dial motif: forming / hunting on loading → lock-in on reveal, with the same position, size & geometry — no jump between screens.
⤳Cinematic transitions between screens — never a hard cut.
●Leading dot appears only during motion; at rest the arc is clean.
06

Layout

⌂Headers float / pin. Always a custom back control — never the native iOS back button (its long-press menu leaks previous screen titles).
▭Scroll content has a bottom inset so the last row clears the floating bottom nav.
⇅Hide scroll indicators everywhere.
⊤Form content is top-anchored; the footer (Continue + Back) is pinned.
›Discoverability: tappable rows get a trailing chevron; editable figures get a subtle edit cue.
▦Same horizontal screen padding (L · 24px) on every screen — content never reflows its left edge.
